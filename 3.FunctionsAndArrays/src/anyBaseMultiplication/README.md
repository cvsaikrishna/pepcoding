# Any Base Multiplication 

1. You are given a base b.
2. You are given two numbers n1 and n2 of base b.
3. You are required to multiply n1 and n2 and print the value.

## Input Format
A base b
A number n1
A number n2
## Output Format
A number of base b equal in value to n2 * n1.

## Constraints
2 <= b <= 10 <br>
0 <= n1 <= 10000 <br>
0 <= n2 <= 10000 <br>
## Sample Input
5 <br>
1220 <br>
31 <br>
## Sample Output
43320
