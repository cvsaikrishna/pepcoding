# Any Base Addition

1. You are given a base b.
2. You are given two numbers n1 and n2 of base b.
3. You are required to add the two numbes and print their value in base b.
## Input Format
A base b
A number n1
A number n2
## Output Format
A number representing the sum of n1 and n2 in base b.

## Constraints
2 <= b <= 10 <br>
0 <= n1 <= 256 <br>
0 <= n2 <= 256 <br>
## Sample Input
8 <br>
777 <br>
1 <br>
Sample Output
1000
